# Student Management System

A Python based Student Management System using Object-Oriented Programming (OOP).

## Features

- Add new student record
- View all students
- Search student by ID
- Update student marks
- Delete student record
- Display student with highest marks

## Technologies Used

- Python
- Object-Oriented Programming (OOP)

## Project Structure

Project.py - Main application file

## How to Run

1. Download the repository
2. Run:

python Project.py
